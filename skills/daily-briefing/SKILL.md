# Daily Briefing

TODO: Implement in Phase 8.

This skill triggers when:
- The user says "morning briefing" or "what's on today"
- The scheduled morning briefing time arrives (configurable, default 08:00)
- The user opens the dashboard for the first time after waking up

The briefing includes:
- Calendar events for the day
- Unread messages and notifications summary
- Overnight worker activity (if any tasks ran while the user slept)
- Key reminders from memory
- Weather (if configured)
